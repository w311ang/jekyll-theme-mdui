## 准备工作（「　 」表示必要）

- 准备安装密钥
- 使用软碟通将 ISO 光盘映像刻录到 U 盘
- BIOS 启用虚拟化
- 导出 Chrome 书签（或前往 chrome://version 将整个用户资料备份）
- 使用 Total Uninstall 备份完整插件的 VB。
- Total Uninstall 导出已安装程序

### 配置分区（非UEFI引导可跳过）

1. 打开 DiskGenius*（推荐使用微 PE）*
2. 将需要安装 Windows 的硬盘转换为 GUID 格式*（**转换有风险，建议备份后再转换。**）*
3. 选中需要安装 Windows 的硬盘的空闲区域
4. 右键建立一个 100MB 的 EFI system partition 分区
5. 再建立一个 NTFS 分区

## 开始安装

- 输入安装密钥后继续安装*（**不输入可能会导致激活失败**）*
- 重启时进入 PE 设置安装盘为 NTFS 压缩并重命名其盘标为 **系统**

### OOBE（初始化界面）

- 不设置账户的密码
